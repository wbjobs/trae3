## 1. 产品概述

智能票据识别系统是一个前后端分离的 AI 应用，旨在通过图像分割与信息抽取技术，自动识别并结构化各类票据（发票、收据、报销单等）中的关键信息，替代人工录入，大幅提升财务处理效率。

- **目标用户**：企业财务人员、报销审核人员、会计从业人员
- **核心价值**：将票据图像自动转化为结构化数据，减少 90% 以上人工录入时间，降低出错率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 上传票据、查看提取结果、导出数据 |
| 管理员 | 管理员邀请 | 用户管理、系统配置、数据统计 |

### 2.2 功能模块

1. **票据上传页**：拖拽/点击上传票据图像，支持批量上传，实时预览与裁剪
2. **识别结果页**：展示 AI 提取的票据字段，支持在线校验与手动修正
3. **票据管理页**：票据数据列表、搜索筛选、详情查看、数据导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 票据上传页 | 拖拽上传区 | 支持拖拽、点击上传，限制格式（JPG/PNG/PDF），单文件最大 10MB，最多同时上传 20 张 |
| 票据上传页 | 上传进度条 | 实时显示上传与识别进度，支持取消上传 |
| 票据上传页 | 图像预览 | 上传后右侧展示缩略图预览，支持放大查看原图 |
| 票据上传页 | 识别状态 | 显示待识别、识别中、识别成功、识别失败四种状态 |
| 识别结果页 | 字段提取展示 | 以卡片形式展示提取结果：发票代码、发票号码、开票日期、金额、税额、价税合计、销售方、购买方等 |
| 识别结果页 | 原图对照 | 左侧显示原图并标注分割区域，右侧显示提取结果，点击字段高亮对应区域 |
| 识别结果页 | 手动修正 | 每个字段可点击编辑，修正后自动保存并标记为"已校验" |
| 识别结果页 | 置信度标注 | 每个字段旁显示 AI 置信度，低于阈值的字段用警告色标记 |
| 票据管理页 | 数据列表 | 表格展示所有票据记录，支持分页，显示关键字段、识别状态、创建时间 |
| 票据管理页 | 搜索筛选 | 按发票号码、日期范围、金额区间、识别状态筛选 |
| 票据管理页 | 详情查看 | 点击行展开详情弹窗，显示完整提取结果与原图 |
| 票据管理页 | 数据导出 | 支持导出为 CSV / Excel 格式 |

## 3. 核心流程

用户上传票据图像 → 前端校验格式与大小 → 后端接收并存储图像 → AI 图像分割识别字段 → 信息抽取与结构化 → 结果存入数据库 → 前端展示提取结果 → 用户校验修正 → 修正后数据更新入库

```mermaid
flowchart TD
    "A[用户上传票据图像]" --> "B[前端校验格式/大小]"
    "B" --> "C[后端接收图像并存储]"
    "C" --> "D[AI 图像分割处理]"
    "D" --> "E[信息抽取与结构化]"
    "E" --> "F[结果写入票据数据库]"
    "F" --> "G[前端展示提取结果]"
    "G" --> "H{用户是否修正?}"
    "H" -->|"是"| "I[修正后数据更新入库]"
    "H" -->|"否"| "J[完成识别流程]"
    "I" --> "J"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深靛蓝 (#1B2A4A) 搭配琥珀金 (#F5A623) 强调色，传递专业与智能感
- **辅助色**：冷灰 (#F4F6F9) 背景、薄荷绿 (#34C759) 成功状态、珊瑚红 (#FF3B30) 错误状态
- **按钮风格**：圆角 8px，主按钮深靛蓝填充，次按钮边框描边，悬停时轻微上移 + 阴影加深
- **字体**：标题使用 Noto Serif SC（思源宋体），正文使用 Noto Sans SC（思源黑体），数据字段使用 JetBrains Mono 等宽字体
- **布局风格**：左侧窄导航 + 右侧宽内容区，卡片式模块布局
- **图标风格**：线性图标，2px 描边，搭配微妙动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 票据上传页 | 拖拽上传区 | 虚线边框区域，中央图标+文字提示，拖入时边框高亮动画，背景渐变 #F4F6F9 → #EBF0F7 |
| 票据上传页 | 上传进度条 | 细长进度条，琥珀金填充，百分比文字右对齐，完成时变为薄荷绿 |
| 票据上传页 | 图像预览 | 圆角卡片，右上角状态徽章，悬停放大镜图标覆盖层 |
| 识别结果页 | 字段提取展示 | 两栏卡片布局，标签深靛蓝、值暗色，低置信度字段左侧橙色竖条标记 |
| 识别结果页 | 原图对照 | 左图右文分栏，图像上叠加半透明色块标注区域，点击字段时对应区域闪烁 |
| 识别结果页 | 手动修正 | 点击字段值变为输入框，保存按钮内联，修正后绿色勾标记 |
| 票据管理页 | 数据列表 | 斑马纹表格，表头固定，行悬停浅蓝背景，状态列彩色圆点标识 |
| 票据管理页 | 搜索筛选 | 顶部搜索栏 + 下拉筛选器，深靛蓝描边输入框 |
| 票据管理页 | 详情弹窗 | 居中弹窗，毛玻璃遮罩，原图与数据双栏对照 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1024px
- 平板适配：内容区单栏堆叠，侧边导航折叠为汉堡菜单
- 移动端：精简表格为卡片列表，上传区全宽
