## 1. 产品概述

战棋沙盘推演系统——一套客户端/服务端分离的兵棋推演平台，客户端提供沉浸式沙盘地图可视化与指挥操作界面，服务端承载战情演算引擎、实时数据同步与对局存档，面向军事爱好者、兵棋玩家及教学推演场景。

- 核心目标：实现可交互的多人实时兵棋推演，支持回合制/即时制战情演算、单位操控、战报回放
- 目标用户：兵棋爱好者、军事推演教学人员、策略游戏玩家

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 指挥官（玩家） | 邀请码注册 | 创建/加入对局、操控己方单位、查看战报 |
| 观察者 | 开放注册 | 观看对局实况、查看历史战报 |
| 管理员 | 后台指派 | 管理地图/剧本、管理用户、查看系统日志 |

### 2.2 功能模块

1. **沙盘大厅页**：对局列表、创建对局、快速匹配、历史战绩入口
2. **沙盘对局页**：地图画布、单位操控面板、战情时间轴、实时通讯
3. **战报复盘页**：回合回放、关键事件标注、数据统计图表
4. **剧本管理页**：剧本编辑器、地图编辑器、单位模板库

## 3. 核心流程

玩家进入大厅 → 创建或加入对局 → 进入沙盘对局页面 → 回合开始：部署阶段 → 移动阶段下达指令 → 服务端演算战斗结果 → 推送结算数据 → 客户端播放战况动画 → 进入下一回合 → 达成胜利条件 → 对局结束 → 生成战报

    "结算阶段" --> "是否达成胜利条件"

    "对局结束" --> "生成战报存档"

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深墨绿 (#0A1F1A) + 战术沙色 (#C4A265) + 警示红 (#D44B3F)
- **辅助色**：地图蓝 (#2B5C8A)、通信绿 (#4CAF50)、数据灰 (#6B7B8D)
- **按钮风格**：微圆角 (4px)，军事风格硬朗线条，主操作按钮带微妙渐变与按压效果
- **字体**：标题使用 Noto Serif SC（庄重感），正文使用 Noto Sans SC（清晰可读），数据/代码使用 JetBrains Mono
- **布局风格**：左侧地图画布占主体 (70%)，右侧操作面板 (30%)，顶部状态栏，底部时间轴
- **图标风格**：线性军事图标风格，统一2px描边，配合语义颜色

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 沙盘大厅 | 对局列表 | 卡片网格布局，每张卡片含地图缩略图、对阵信息、状态标签，深色背景 |
| 沙盘大厅 | 创建对局 | 模态对话框，剧本选择下拉、参数表单、邀请输入框 |
| 沙盘对局 | 地图画布 | Canvas全屏渲染，六角格叠加层，缩放控件右下角，迷你地图左下角 |
| 沙盘对局 | 单位操控 | 右侧可折叠面板，单位头像+属性条，行动按钮组（移动/攻击/待机） |
| 沙盘对局 | 战情时间轴 | 底部水平时间轴，阶段节点高亮，回合数显示 |
| 沙盘对局 | 实时通讯 | 右下角悬浮聊天窗口，半透明背景，消息气泡 |
| 战报复盘 | 回放控制 | 底部播放控制栏，进度条+速度选择，回合跳转下拉 |
| 战报复盘 | 数据统计 | 右侧Tab切换：损失饼图、控制区折线图、资源柱状图 |

### 4.3 响应式策略

- 桌面优先设计，最低支持 1280×720
- 平板端：操作面板改为底部抽屉，地图全屏
- 移动端：暂不支持，提示使用桌面端

### 4.4 动效设计

- 地图加载：从中心向外渐显六角格
- 单位移动：沿路径逐格平滑过渡
- 战斗结算：爆炸粒子特效 + 伤害数字弹出
- 回合切换：阶段名称滑入动画
- 面板切换：抽屉式滑入/滑出
