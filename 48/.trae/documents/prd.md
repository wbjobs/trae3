## 1. 产品概述

巡检智能缺陷识别平台——面向工业巡检场景的 AI 辅助系统，通过上传巡检图片，自动完成图像预处理、AI 推理、缺陷分类，并结合向量库实现相似缺陷快速检索与归类，大幅提升巡检效率与缺陷管理精度。

- 目标用户：电力/油气/制造等行业的巡检工程师、质量管理人员、运维决策者
- 核心价值：将人工巡检识别流程自动化，降低漏检率，实现缺陷的标准化分类与历史追溯

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 巡检员 | 管理员分配账号 | 上传图片、查看识别结果、提交确认 |
| 质量管理员 | 管理员分配账号 | 审核缺陷结果、分类修正、统计分析 |
| 系统管理员 | 系统初始化 | 用户管理、模型配置、系统设置 |

### 2.2 功能模块

1. **巡检工作台页面**：图片上传区域、实时识别进度、识别结果卡片、缺陷标注预览
2. **缺陷库管理页面**：缺陷分类列表、向量相似检索、缺陷详情与历史对比
3. **统计分析页面**：缺陷分布图表、趋势分析、巡检报告导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 巡检工作台 | 图片上传区 | 支持拖拽/批量上传图片，实时显示上传进度与缩略图预览 |
| 巡检工作台 | AI 识别面板 | 展示推理进度、缺陷检测框标注、置信度评分、分类标签 |
| 巡检工作台 | 结果确认区 | 巡检员对识别结果进行确认/修正，添加备注说明 |
| 缺陷库管理 | 分类列表 | 按缺陷类型（裂纹/锈蚀/变形/缺失等）分类浏览，支持筛选排序 |
| 缺陷库管理 | 向量检索 | 输入缺陷描述或上传参考图，基于向量相似度检索历史缺陷 |
| 缺陷库管理 | 详情对比 | 查看缺陷详情、相似缺陷对比、历史维修记录 |
| 统计分析 | 分布图表 | 按类型/严重程度/位置等维度展示缺陷分布饼图/柱状图 |
| 统计分析 | 趋势分析 | 时间维度缺陷趋势折线图，支持日/周/月切换 |
| 统计分析 | 报告导出 | 生成巡检报告 PDF，支持自定义时间范围与筛选条件 |

## 3. 核心流程

用户上传巡检图片 → 前端发送至后端 → 图像预处理模块（缩放/增强/去噪）→ AI 推理模块（目标检测 + 特征提取）→ 缺陷分类模块（类型判定 + 严重度评估）→ 向量库写入（特征向量存储）→ 结果返回前端展示 → 用户确认/修正 → 数据持久化存储

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深邃蓝 (#0F172A) + 科技青 (#06D6A0)，传达工业可靠感与智能科技感
- 辅助色：警示橙 (#FF6B35)、危险红 (#EF476F)、安全绿 (#06D6A0)
- 按钮：圆角微弧 (rounded-lg)，带微妙阴影与 hover 发光效果
- 字体：标题用 Noto Sans SC Bold，正文用 Noto Sans SC Regular
- 布局：左侧固定导航栏 + 右侧内容区，卡片式模块布局
- 图标风格：线性描边图标，统一 2px 描边宽度

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 巡检工作台 | 图片上传区 | 虚线边框拖拽区，上传动画，缩略图网格，进度条 |
| 巡检工作台 | AI 识别面板 | 左侧原图+检测框叠加，右侧缺陷列表卡片，置信度进度条 |
| 巡检工作台 | 结果确认区 | 缺陷卡片列表，确认/修正按钮，备注输入框 |
| 缺陷库管理 | 分类列表 | 左侧分类树，右侧缺陷卡片网格，筛选器 |
| 缺陷库管理 | 向量检索 | 搜索输入框，图片上传按钮，相似度排序结果列表 |
| 缺陷库管理 | 详情对比 | 大图预览，并排相似缺陷对比，属性面板 |
| 统计分析 | 分布图表 | 饼图 + 柱状图，hover 详情弹窗，图例筛选 |
| 统计分析 | 趋势分析 | 折线图 + 面积图，时间粒度切换，数据下钻 |
| 统计分析 | 报告导出 | 日期范围选择器，筛选条件，预览 + 导出按钮 |

### 4.3 响应式设计

- 桌面优先设计，核心操作区 1280px+ 最佳体验
- 中等屏幕 (768px-1280px)：侧边栏折叠为图标模式，内容区自适应
- 移动端暂不作为核心场景，确保基础可用即可
