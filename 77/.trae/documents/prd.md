## 1. 产品概述

实时监测与异常识别系统，用于接收流式监测数据、计算关键指标、自动识别异常模式，并通过可视化图表展示数据。解决工业设备、系统运维等场景下的实时监控需求，帮助用户快速发现并定位异常问题。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 系统管理员 | 默认配置 | 查看所有监测数据、管理告警规则、导出报表 |
| 普通用户 | 管理员分配 | 查看授权的监测指标、设置个人筛选条件 |

### 2.2 功能模块

1. **实时监测面板**：关键指标概览卡片、实时数据流展示、系统健康状态
2. **图表可视化**：多维度折线图、柱状图、仪表盘、异常标记高亮
3. **数据筛选**：时间范围选择、指标类型筛选、设备/来源筛选、阈值过滤
4. **异常告警**：异常事件列表、告警级别标识、异常详情查看、告警趋势统计
5. **指标统计**：均值、峰值、波动率、分位数等统计指标计算

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 监测面板 | 概览卡片 | 展示关键指标实时值、变化趋势箭头、健康状态指示 |
| 监测面板 | 实时图表 | 流式数据实时更新折线图，支持多指标叠加对比 |
| 数据筛选 | 筛选栏 | 时间范围选择器、指标多选框、设备来源下拉框 |
| 数据筛选 | 高级筛选 | 阈值过滤、异常状态过滤、数据聚合粒度选择 |
| 异常告警 | 告警列表 | 按时间倒序的异常事件，含级别、类型、持续时间 |
| 异常告警 | 告警统计 | 告警级别分布饼图、趋势柱状图、Top异常指标 |
| 指标详情 | 历史趋势 | 选中指标的历史数据图表、支持缩放和时间窗选择 |
| 指标详情 | 统计面板 | 展示最大值、最小值、平均值、标准差、波动率 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰蓝 `#0f172a`（背景），科技青 `#06b6d4`（主强调），告警红 `#ef4444`（错误），警戒橙 `#f59e0b`（警告），正常绿 `#10b981`（成功）
- **字体**：JetBrains Mono（数字展示），Noto Sans SC（中文正文）
- **按钮风格**：方角、细边框、悬停发光效果
- **布局风格**：网格化卡片布局、左侧导航栏+顶部状态栏+主内容区三栏结构
- **视觉风格**：工业科技风，深色主题，发光边框，数据大屏质感，适当的毛玻璃效果
- **图标风格**：线性图标，配合发光效果，使用 lucide-react

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 监测面板 | 概览卡片 | 发光边框、渐变背景、实时数值跳动动画、趋势箭头指示 |
| 监测面板 | 实时图表 | ECharts折线图、数据点闪烁动画、异常区域红色高亮 |
| 数据筛选 | 筛选栏 | 紧凑布局、下拉选择器、日期范围选择、标签式指标切换 |
| 异常告警 | 告警列表 | 行背景色区分告警级别、闪烁动画提示新告警、展开查看详情 |
| 异常告警 | 统计面板 | 环形进度图展示告警分布、数字滚动动画 |
| 指标详情 | 统计面板 | 网格布局、关键指标大号字体、辅助指标小号灰色字体 |

### 4.3 响应式

- 桌面端优先设计，1920px为基准
- 适配1280px以上屏幕，内容区采用流式布局
- 卡片在中等屏幕自动换行，小屏幕单列堆叠
- 图表容器使用百分比宽度，自动适应父容器

### 4.4 动画效果

- 页面加载：卡片渐入+上移动画，按顺序延迟出现
- 数据更新：数值平滑过渡动画，新数据点标记闪烁
- 异常发生：告警条目从右滑入，红色边框脉冲动画
- 悬停效果：卡片微放大、阴影加深、边框发光增强
- 切换筛选：图表数据平滑过渡，无生硬跳变
